ezvi.funcmodule.ez_spawn(argv, instructions, master_read=<function ez_read>, stdin_read=<function ez_read>)¶

Spawns a process To spawn the process. Heavily inspired from Python’s pty module. ez_spawn() should only be used once per file that needs to be edited. This function can write a whole text file following the instructions dictionary and executing those instructions using ez_copy().

Parameters

argv (tuple) – First element is the program to run. Other elements

are the arguments passed to the program. :type instructions: list :param instructions: Contains all the instructions that will be passed to Vi. Its contents should have been created using the “Vi tools”. :type master_read: function :param master_read: The function that will be used to read info from the master’s file descriptor. :type stdin_read: function :param stdin_read: The function that will be used to read info from the master’s file descriptor.

Return type

int

Returns

Exit status
